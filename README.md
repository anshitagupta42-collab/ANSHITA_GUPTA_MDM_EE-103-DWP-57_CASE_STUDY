# ANSHITA_GUPTA_MDM_EE-103-DWP-57_CASE_STUDY
This project is a frontend-only food delivery website inspired by Zomato, developed using HTML, CSS, and JavaScript. The website is designed to provide users with an interactive and visually appealing platform to explore restaurants, browse menus, search for food items, and manage a cart, similar to a real online food ordering system.
